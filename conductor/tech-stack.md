# Technology Stack

## Frontend
- **Framework:** [Next.js](https://nextjs.org/) (App Router) - For a modern, SEO-friendly, and performant web application.
- **Library:** [React](https://react.dev/) (v19) - For building interactive and reusable user interface components.
- **Language:** [TypeScript](https://www.typescriptlang.org/) - For enhanced developer productivity and code reliability through static typing.
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) - For rapid, utility-first UI development and a consistent design system.
- **State Management:** [TanStack Query](https://tanstack.com/query/latest) (React Query) - For efficient server-state management, caching, and data synchronization.
- **Icons:** [Hugeicons](https://hugeicons.com/) - For a clean and modern icon set.
- **UI Components:** [Radix UI](https://www.radix-ui.com/) (via @base-ui/react) - For accessible and unstyled UI primitives.

## Backend & Services
- **Backend-as-a-Service:** [Firebase](https://firebase.google.com/) - Utilizing Authentication for user accounts and Firestore for real-time database capabilities.
- **Server-Side Operations:** Firebase Admin SDK - For secure, privileged backend operations.

## Data Sources & APIs
- **Media Metadata:** [TMDB API](https://developer.themoviedb.org/) - The primary source for movie and TV show information, posters, and backdrops.
- **Tracking & Social:** [Trakt.tv API](https://trakt.docs.apiary.io/) - For logging watch history, syncing viewing progress, and community features like reviews.

## Development Tools
- **Package Manager:** [pnpm](https://pnpm.io/) - For fast and disk-efficient dependency management.
- **Linting & Formatting:** [ESLint](https://eslint.org/) and [Prettier](https://prettier.io/) - To maintain code quality and consistent styling.
